# 3階層 業務フロー図ジェネレーター

業務マニュアル・手順書・会議メモなどのテキストを、A4横向きのPDF業務フロー図に変換するWebアプリです。  
単一HTMLファイルで動作し、インストール不要・サーバー不要で使えます。

---

## 主な機能

- **3階層フロー構造**：大フロー / 中フロー / 小フロー の階層表示
- **Markdown入力**：所定の記法で記述したMarkdownを貼り付けるだけでフロー図を生成
- **PDF・ZIP出力**：A4横（1123×794px）のページ単位で書き出し
- **自動ページ分割**：内容量に応じてページを自動生成
- **ボトルネック表示**：フロー名末尾に `[ボトルネック]` を付けると視覚的に強調
- **サマリー / 作業ステップ / 備考 / 関連リンク**：小フロー配下に詳細情報を記載可能
- **元に戻す / やり直し（Undo/Redo）**
- **ドラッグ＆ドロップによる並び替え**
- **データ消失防止**：未保存変更がある場合はページ離脱を警告

## 使い方

### 1. アプリを開く

`index.html` をブラウザで直接開くか、公開URLにアクセスします。

### 2. Markdownを入力する

左ペインのテキストエリアに、以下の記法でMarkdownを貼り付けます。

```markdown
# タイトル
date: 2025/01/01
author: 担当者名

## 大フロー名

### 中フロー名

- 小フロー名
  > サマリー（概要）
  1. ステップ1のタイトル
     ステップ1の説明文
  2. ステップ2のタイトル
     ステップ2の説明文
  ---
  備考: 補足事項をここに書く
  🔗 -> [関連フロー名](フローID)

- ボトルネックのある小フロー [ボトルネック]
```

### 3. フロー図を確認・出力する

- 右ペインにリアルタイムでプレビューが表示されます
- 「PDFダウンロード」または「ZIPダウンロード」ボタンで出力できます

---

## Markdown記法リファレンス

| 記法 | 意味 |
|------|------|
| `# タイトル` | 帳票タイトル（1行目） |
| `date: YYYY/MM/DD` | 作成日 |
| `author: 名前` | 作成者 |
| `## 大フロー名` | 大フロー |
| `### 中フロー名` | 中フロー |
| `- 小フロー名` | 小フロー |
| `[ボトルネック]` | 行末に付けると強調表示 |
| `> テキスト` | サマリー（小フロー直下） |
| `1. ステップ名` | 作業ステップ（小フロー直下） |
| `---` | 備考・リンクの区切り線 |
| `備考: テキスト` | 備考（`---`の後） |
| `🔗 -> [名称](ID)` | 関連フローへのリンク（`---`の後） |

## AIによるMarkdown自動生成（オプション）

業務マニュアルや手順書のテキストを貼り付けると、このアプリ用のMarkdownに自動変換できます。

### システムプロンプト（コアスキル定義）

`共通コアスキル定義.md` をClaude等のAIにシステムプロンプトとして設定します。  
このスキル定義により、AIが入力文書を3階層Markdown形式へ正規化します。

**主な変換ルール：**
- 業務フェーズ・部門単位 → 大フロー
- 工程群・処理カテゴリ → 中フロー  
- 実作業・確認・入力・承認 → 小フロー
- 承認待ち・属人化・手戻り多発などが明示されている場合のみボトルネックを付与
- IDが明示されていない場合は関連リンクを出力しない

### ユーザープロンプトのテンプレート

`推奨ユーザープロンプト.txt` をベースに、以下のように入力します。

```
以下の文書を、3階層業務フロー図ジェネレーター用Markdownへ変換してください。

# 変換モード
- mode: strict_importable_md

# 追加指定
- リンク先IDが明示されていない限り、関連リンクは出力しない
- summary / steps / notes は出力してよい
- 不明な情報は省略する

# 入力文書
（ここに原文を貼り付け）
```

生成されたMarkdownをそのままアプリに貼り付けてフロー図を作成できます。
